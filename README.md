# staticweb

A simple static web page built with HTML and CSS.

## Features

- Clean and modern design
- Responsive layout that works on mobile and desktop
- Semantic HTML5 structure
- CSS styling with gradient header and smooth transitions

## How to View

1. Clone this repository:
   ```bash
   git clone https://github.com/vuphama/staticweb.git
   ```

2. Open the `index.html` file in your web browser:
   - Double-click the file, or
   - Right-click and select "Open with" your preferred browser, or
   - Use a local web server for a more realistic experience

## Using a Local Web Server

For a better development experience, you can use a local web server:

### Python
```bash
python -m http.server 8000
```

### Node.js (with npx)
```bash
npx http-server
```

Then open `http://localhost:8000` in your browser.
